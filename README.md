# ExamDistributionTool

## Requirements
DotNet Core Runtime 5.0 Required.

## Command line tool

Argument: the path of the input file

## Input file format

Example:

```
[
  {
    "name": "Nome1",
    "date" : ["2021-01-01", "2021-12-31"]
  },
  {
    "name": "Nome2",
    "date" : ["2021-04-01", "2021-05-31"]
  },
  {
    "name": "Nome3",
    "date" : ["2021-07-01", "2021-11-30"]
  }
]
```
